HOW TO ADD AN IMAGE:
1. Place the image file in this folder (images/)
2. Find the matching <!-- EDIT --> comment in the HTML file
3. Replace the placeholder with: <img src="images/YOUR-FILE.jpg" alt="Description" />

PHOTOGRAPHY GALLERY:
For best performance, use two versions of each photo:
  photo-1-thumb.jpg     → Smaller/compressed version (400-600px wide)
  photo-1-full.jpg      → Full quality version (for lightbox)

  photo-2-thumb.jpg / photo-2-full.jpg
  photo-3-thumb.jpg / photo-3-full.jpg
  ... and so on up to as many photos as you have

In the HTML, reference them like:
  <img class="gallery-thumb"
       src="images/photo-1-thumb.jpg"
       data-full="images/photo-1-full.jpg"
       alt="Your caption here" />


═══════════════════════════════════════════════════════════
IMAGE TIPS
═══════════════════════════════════════════════════════════
- Supported formats: .jpg  .jpeg  .png  .webp  .gif
- Keep file sizes reasonable: compress photos at tinypng.com
- Profile photos: square crop works best for circle display
- Book covers: portrait ratio (2:3) looks best
- Travel photos: landscape (16:9 or wider) looks best
- Gallery photos: any ratio works (they're cropped to square)
- Award scans: good lighting, straight scan, high contrast
- Organization logos: .png with transparent background is ideal

COPYRIGHT REMINDER:
All photos you upload should be your own original work,
or images you have permission to use. Your own research
photos, field work images, and personal photographs are safe.
